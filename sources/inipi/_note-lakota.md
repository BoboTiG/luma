`````{admonition} Notes à propos de la prononciation
:class: toggle

Dans l'usage des dictonnaires *lakota* modernes, la prononciation s'entend comme suit :

````{grid} 2

```{grid-item}
- **c** → "tch"
- **ġ** → "dj"
- **ḣ** → "r"
```

```{grid-item}
- **ǩ** → "cr"
- **š** → "ch"
- **ṫ** → "tr"
```

````

Les femmes prononcent les finales **o** en "é", exemples :
- *ce'kiyay**o*** → *ce'kiyay**e***
- *yankel**o*** → *yankel**e***
`````
